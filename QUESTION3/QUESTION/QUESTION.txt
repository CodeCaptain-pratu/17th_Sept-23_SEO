QUESTION:What are the Benefits of Using Meta Tags
Answer:The benifits of using meta tags are as follows -
1) SEO (Search Engine Optimization):Meta tags such "description" and "Keyword"
tags, can help to improve a webpage's visiblity on search engine 
result pages(SERP).
2) Social Media Sharing - Meta tags like the open graph protocol tags
allow developer to control how web pages are displayed when shared 
on social media like facebook twitter and linkedIn.
3) Viewport control - The Viewport meta tag is crucial for responsive 
web design.
4) HTTP-Equiv tags - Meta tags with http-equiv attribute (e.g., 
<meta http-equiv="refresh">) enable specific HTTP header 
functionalities, like refreshing or redirecting the page after a
certain time.
